# Omni-Server Frame

A modular, production-ready server framework for controlling AI agents (specifically Kilo Code) via the Agent Client Protocol (ACP). Built with Node.js/TypeScript.

## Status

**Phase 1: Project Setup** - In Progress

## Quick Start

```bash
npm install
npm run dev
```

## Architecture

```
Client (Browser) ←→ Omni-Server Frame ←→ Kilo ACP (subprocess)
```
